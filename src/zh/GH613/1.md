- GH613 DSP数字2.0立体声TWS功放板，板载DSP采用高性能32位处理器，ADC、DAC采样率支持 96kHz 输入输出。
- 音频算法解码：MP2、MP3、MP4、WAV、AIF、AIFC；编码支持：MP2 / MP3、IMA-ADPCM。
- 音效算法支持：降噪、3D 立体声环绕、PEQ 均衡器、DRC 动态限幅器，支持在线调音，支持在线 OTA 固件升级。
- 输入支持：板载蓝牙解码输入，支持 USB 无损声卡输入，支持 U 盘输入，支持 AUX 3.5mm 模拟音频输入。
- 双模蓝牙 V5.4，兼容蓝牙 V4.2 和 V2.1+EDR，支持蓝牙 Piconet 和 Scatternet 组网协议，最大发射功率 10dBm，支持 A2DP / ACRCP / HFP / OPP / HID / SPP / PBAP / GATT / SM 等协议。
- 电源功率：支持 6–19V 电压输入，电流 10–50A。
- 功率输出：输出为 2.0 声道，高功率输出（标准抗阻 3 欧）为 80W × 2。
- 配件支持：红外遥控、按键板、尾插接口板，内置 2.4G 遥控接收。
